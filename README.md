# Визуализация дифракции на решётке

## Общее описание

Рассматривается 1D дифракция в схеме, изображенной на рисунке ниже. Источник находится в фокальной плоскости линзы 1, экран - в фокальной плоскости линзы 2. Линзы считаются идеальными. Программа строит графики интенсивности и видности дифракционной картины на экране для заданных параметров системы.

### Параметры системы

| Параметр | Значение по умолчанию | Описание |
|----------|-----------------------|----------|
| $a$ | 0 мкм | Вертикальный размер источника |
| Форма источника (`uniform`/`guassian`) | `uniform` | Вид плотности интенсивности источника |
| $\lambda_0$ | 550 нм | Длина волны источника |
| $\Delta\lambda$ | 0 нм | Ширина спектра источника |
| Спектр (`mono`/`gaussian`) | `mono` | Форма спектра |
| $F_1$ | 100 мм | Фокусное расстояние первой линзы |
| $F_2$ | 100 мм | Фокусное расстояние второй линзы |
| $N$ | 5 | Количество щелей решётки |
| $b$ | 20 мкм | Ширина щелей |
| $d$ | 50 мкм | Период решётки |
